# Music-Library-System
A simple and efficient music library management system built in C. 

Overview

CPlayer Music Library is a simple console-based application written in C that allows users to manage a collection of songs. Users can add, view, search, and organize songs through an easy-to-use menu-driven interface.

Features
Add new songs to the library
View all songs
Search songs by title
Store song information in a text file
Simple and user-friendly menu system
Technologies Used
C Programming Language
File Handling (fopen, fclose, fprintf, fscanf)
String Handling (strcmp, strcpy, strlen)
Arrays and Structures
